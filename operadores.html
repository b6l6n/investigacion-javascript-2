OPERADORES

¿Que es un operadore?

Un operador es un símbolo especial o palabra clave que se utiliza en
programación para realizar operaciones sobre valores o variables. En
JavaScript, los operadores se utilizan para asignar valores, comparar
valores, y realizar operaciones aritméticas, lógicas y bitwise.

Lista de Operadores de Comparación:

  • 1.== (Igual )
  • 2.=== (Igual estrictamente )
  • 3.!= (No igual )
  • 4.!== (No igual estrictamente )

1.== (Igual )

Definicion: El operador == se utiliza para comparar solo los valores de dos elementos.
Sin embargo, no le importa el tipo de esos elementos (si son números, cadenas,
booleanos, etc.). Este operador solo verifica si los valores son los mismos,
aunque sean de tipos diferentes. Si son iguales, devuelve true, y si no, devuelve false.

2.=== (Igual estrictamente )

Definicion: El operador === es más estricto que ==. Este operador compara tanto el valor como el
tipo de los elementos. Esto significa que, para que dos elementos sean iguales con ===,
su valor y su tipo deben coincidir exactamente.

EJEMPLO:

3.!= (No igual )

Definicion: El operador != se usa para verificar si dos valores no son iguales. Al igual que ==,
!= solo compara los valores y no le importa el tipo de esos valores. Si los valores son
diferentes, devuelve true; si son iguales, devuelve false.

EJEMPLO:

4.!== (No igual estrictamente )

Definicion: El operador !== es el opuesto de ===. Este operador compara tanto el valor como el tipo de los
elementos, y devuelve true si alguno de los dos (valor o tipo) es diferente. Si tanto el valor
como el tipo son iguales, devuelve false.

OPERADORES LOGICOS

Los operadores lógicos se utilizan en programación para combinar o manipular valores booleanos

Los operadores principales son:

  • AND &&
  • OR ||
  • NOT

1.AND &&

Definicion: El operador AND (&&) devuelve true solo si ambas condiciones son verdaderas.
En cualquier otro caso, el resultado será false.

2.OR ||

Definicion: El operador OR (||) devuelve true si al menos una de las condiciones es verdadera. Solo
devuelve false cuando ambas condiciones son falsas.

3. NOT !

Definicion: El operador NOT (!) se usa para invertir el valor de una condición. Si es true, se
convierte en false, y si es false, se convierte en true.
